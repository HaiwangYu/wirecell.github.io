virtual WireCell::IWire::vector WireCell::ICell::wires ( ) const
pure virtual

Return a vector of wires which are associated with this cell. The exact association depends on the algorithm followed to produce the cells. Most typical is that the vector will hold one wire from each plane which closely intersects the cell boundary. In general, the content and ordering is not specified.

Implemented in WireCell::BoundCell, and WireCell::SimpleCell.
