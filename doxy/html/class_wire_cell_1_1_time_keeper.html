Detailed Description

A helper class to give some time keeping.

Use like

TimeKeeper tk("starting"); ... cerr << tk("starting long calculation....") << endl; do_long_calculation(); cerr << tk("...done") << endl;; ... cerr << tk.summary() << endl;
