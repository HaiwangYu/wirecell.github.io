Detailed Description

Interface to information about an aggregation of cells which itself is a cell (implements WireCell::ICell interface).

A blob is a cell which is composed of other Cells. This means that one may use a blob to hold other blobs, if warranted.

When you construct IBlobs be mindful of using .ident() numbers which are unique even among the individual ICell constituents.
