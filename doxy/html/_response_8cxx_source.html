245 /*
246  Cold Electronics response function.
247 
248  How was this function derived?
249 
250  1. Hucheng provided a transfer function of our electronics, which
251  is the Laplace transformation of the shaping function.
252 
253  2. We did a anti-Laplace inverse of the shaping function
254 
255  3. The one you saw in the code is basically the result of the inversion
256 
257  - time is time in system of units
258 
259  - gain in units of volts/charge gives peak value of the response to
260  a delta function of current integrating to unit charge.
261 
262  - shaping is the shaping time in system of units
263 
264  - the hard-coded numbers are the result of the inverting the
265  Lapalace transformation in Mathematica.
266 
267  */
